# AdventTools

Unelte software pentru **[Biserica Adventistă de Ziua a Șaptea din România](https://adventist.ro)**.

Dezvoltate de **Samy Balasa** și **Ovidius Zanfir**. Tot ce vezi aici e
open-source, sub licență MIT.

## Ce facem

Construim instrumente practice care ușurează munca administrativă și
organizatorică în biserică: generatoare de documente, automatizări, sisteme
mici care economisesc timp. Fiecare repo aici e un proiect pe care îl folosim
noi sau echipe pe care le sprijinim.

## Proiecte

- **[dgpt-scripts](https://github.com/AdventTools/dgpt-scripts)** — Generator
  de diplome și planșe PPTX pentru campania
  *[Din grijă pentru tine](https://dingrijapentrutine.ro)*.

(Și altele care vor veni.)

## Vrei să contribui?

Te primim cu bucurie. Fie că ai o idee, un bug de raportat, o îmbunătățire,
sau vrei să adaugi un proiect nou:

- **Pentru un proiect existent** → deschide un Issue sau Pull Request pe
  repo-ul respectiv (vezi `CONTRIBUTING.md` în fiecare).
- **Pentru o idee nouă** → deschide un Issue în oricare repo.
- **Pentru a face parte din echipă** → contactează-ne.

Totul e voluntar, pentru folosul comunității.
